what is CSS?

  1. CSS stands for cascading style sheets

  2. CSS describes how HTML elements are to be displayed on screen,paper,or in the other media

  3. Css is not neccesary CSS saves a lot of work.it can control the layout the layout of multiple web pages all at once

  4. External style sheeets are stored in CSS files

CSS can be added to HTML documents in 3 ways

  • inline-by using the style attribute inside HTML element1
  • internal-by using the style element in the head section
  • External-by using a link element to link an external CSS file
  • The most common way to add CSS. is to keep the styles in external CSS files. however,in this tutorial we will use inline and internal styles,because this is easier to demonstrate,and easier for you to try it yourself

When comparing CSS class vs ID, the difference is that CSS class applies a style to multiple elements.ID, on other hand,applies a style to one unqiue element.ID is also special in that you can use a special URL to link directly to an element and its used by JavaScript

In CSS,selection are used to target a specific element or range of elements on a web page.Once an element has been targeted, a style or set of styles can be applied to the element.

There is a wide range of selectors are available.two of the most commonly used are class and ID.Both are used to target elements to which a style should be applied
